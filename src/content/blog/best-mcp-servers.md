# MCP Servers Guide

Model Context Protocol (MCP) servers are transforming how we build AI applications.

## What is MCP?

MCP provides a standardized way for AI models to interact with external tools and data sources.

## Top Free MCP Servers

1. **Filesystem MCP** - Local file operations
2. **GitHub MCP** - Repository management
3. **Database MCP** - Database queries
4. **Brave Search MCP** - Web search capabilities

Setting up MCP servers is straightforward. Most require Docker and basic configuration.
